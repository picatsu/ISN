Notre Projet Principale

Nos Raisons et Objectifs :

  • 1 : Nous avons voulu créer un programme du jeu du Pendu en python pour notre projet commun.
  • 2 : Le site ou tout sera référencé sera sur le logiciel Olympe, un hébergeur en ligne.
  • 3 : Un programme python fonctionnel mais ne pouvant être affiché sur le site sera disponible.
  • 4 : Le pendu comporte certaines règles particulière, comme par exemple l'ajout de niveau de difficulté.
  • 5 : Le site Web comportera aussi un fichier LARP montrant l'algorithme/version simplifier du jeu.
  • 6 : Un fichier HTML, un CSS et un programme Python fonctionnel.

    Decouverte

                 

    1. - Nous avons voulu reprendre le principe du jeu "2048" apparu sur les plates-formes mobiles qui est en ce moment dans le top 10 des applications téléphoniques, en langage python, sans interface graphique, programme fonctionnel et commenté disponible.
    2. - On a repris le principe du jeu mais on a modifié la taille de tableau, programme fonctionnel et commenté disponible.
    3. - Nous avons aussi voulu reprendre en python, le jeu Puissance 4, mais il reste assez compliqué et à ce jour n'est pas encore terminé.
    4. - Un jeu de dés, où les régles sont spéciales et où l'on peut parier des petites sommes, programme fonctionnel et commenté disponible.

    6. - Nous avons aussi developpé un programme du jeu des Allumettes, qui nous tenait à coeur, programme fonctionnel et commenté disponible.
